# Zaylie's Golden Paw Party

A small GitHub Pages-ready birthday page for Zaylie's stuffed puppy adoption party.

## Files

- `index.html` - the party landing page
- `zaylie_baby_girl_golden_paw.gif` - animated GIF
- `golden_paw_adoption_academy_kit.pdf` - printable party games kit
- `zaylie_baby_girl_golden_paw_preview.png` - still preview image

## Use With GitHub Pages

1. Put these files in a GitHub repository.
2. In GitHub, open repository settings.
3. Go to Pages.
4. Publish from the branch that contains `index.html`.

The page will show the animated GIF and link to the printable kit.
